<h2 id="english">English</h2>

A Hugo theme designed exclusively for plain text blogging. It returns the digital reading experience to its purest form by stripping away redundant visual noise.

![Lucid Void Screenshot](images/screenshot.png)

### ✨ The Design Philosophy

The visual framework of this theme is **Lucid Void**, a tailored aesthetic language born from **Lucentia** (the author's personal design ethos). 

It is not a strict implementation of any existing framework, but rather a synthesis of concepts: it borrows the expansive, spatial navigation paradigms of the classic **Metro UI**, and introduces the subtle tactile depth of **Material Design**. These concepts are applied to a strictly monochromatic "void" (black, white, and gray) featuring a highly legible academic serif typographic stack. This absolute quietness allows the customizable accent colors to leap out, injecting life into a plain text environment.

### 🚀 Features
- **Content-First Architecture**: No complex category trees, no over-engineered widgets. Absolute focus on the text.
- **Opinionated Taxonomy**: Employs a flat, Tag-only system for ultimate minimalism.
- **Zero-Dependency**: Vanilla JavaScript only. No jQuery, no bloated libraries.
- **Smart Loading**: Auto-sniffing syntax highlighting CSS and Math (KaTeX) support, loaded per-page on demand.

### ⚙️ Installation & Usage
Inside the folder of your Hugo site run:
```bash
git submodule add [https://github.com/yourusername/hugo-theme-lucidvoid.git](https://github.com/yourusername/hugo-theme-lucidvoid.git) themes/lucidvoid
```

For configuration, please check the `hugo.toml` file inside the `exampleSite` folder of this repository.

<h2 id="中文">中文</h2>

专为纯文本博客打造的 Hugo 主题。通过剔除多余的视觉噪音，让数字阅读体验回归最纯粹的本质。

### ✨ 设计哲学

本主题的视觉框架采用了 **Lucid Void (清透之境)** ， 这是一个脱胎于作者个人设计理念 **Lucentia** 的定制美学语言。

它并非生搬硬套现有的 UI 框架，而是一种理念的融合：它借鉴了经典 **Metro UI** 充满空间感的导航范式，并适度融合了 **Material Design** 的触觉深度。这些理念被置入一个由黑、白、灰构成的“绝对静谧”的单色空间中，并辅以严肃的衬线排版。这种极简的底色，让用户自定义的高亮点缀色得以跃然而出。

### 🚀 核心特性

- **内容至上架构**：没有复杂的分类树和臃肿的小组件，将绝对的焦点还给文字。
- **带有态度的分类学**：极致克制，仅保留扁平化的“标签 (Tags)”系统，彻底废弃“分类 (Categories)”。
- **零依赖与高性能**：纯原生 JavaScript，无 jQuery 等臃肿依赖。按需智能加载代码高亮与数学公式 (KaTeX) 引擎。

### ⚙️ 安装与使用

在你的 Hugo 站点根目录下执行：

Bash

```
git submodule add [https://github.com/yourusername/hugo-theme-lucidvoid.git](https://github.com/yourusername/hugo-theme-lucidvoid.git) themes/lucidvoid
```
