<h2 id="english">English</h2>

A Hugo theme designed for plain text blogging. It returns the digital reading experience to its purest form by stripping away redundant visual noise.

### ✨ The Design Philosophy
The visual framework of this theme is **Lucid Void**, a tailored aesthetic language born from **Lucentia** (the author's personal design ethos). 

It is not a strict implementation of any existing framework, but a synthesis of concepts: the expansive, spatial navigation paradigms of classic **Metro UI**, subtly infused with the tactile depth of early **Material Design**. Set against a strictly monochromatic "void," the theme employs a highly legible academic serif typography stack. This absolute quietness allows your customizable accent colors to leap out, injecting life into a plain text environment.

### 🚀 Features
- **Content-First Architecture**: No complex category trees, no over-engineered widgets. Absolute focus on the text.
- **Academic Typography**: Meticulously tuned serif typography stack for the main body, ensuring a serene and immersive reading experience.
- **Opinionated Taxonomy**: Employs a flat, Tag-only system for ultimate minimalism.
- **Smart & Zero-Dependency**: Vanilla JavaScript only. Auto-sniffing syntax highlighting CSS and Math (KaTeX) support, loaded per-page only when needed.
- **Lightweight Search**: Built-in, extremely fast client-side search powered by native JS.
- **Immersive Viewing**: Native image lightbox functionality for an uninterrupted viewing experience.
- **Community Ready**: Seamless integration with Giscus comments.

### ⚙️ Installation & Usage
Inside the folder of your Hugo site run:
```bash
git submodule add [https://github.com/AstroNomen/hugo-theme-lucidvoid.git](https://github.com/AstroNomen/hugo-theme-lucidvoid.git) themes/lucidvoid
```

For configuration, please check the `hugo.toml` file inside the `exampleSite` folder of this repository.

<h2 id="中文">中文</h2>

为纯文本博客打造的 Hugo 主题。通过剔除多余的视觉噪音，让数字阅读体验回归最纯粹的本质。

✨ 设计哲学
本主题的视觉框架采用了 **Lucid Void (清透虚空)** ， 一个脱胎于作者个人设计实验室 Lucentia 的设计语言。

它主要受到经典 Metro UI 充满空间感的导航范式的启发，并适度加入了 Material Design 的触觉深度。主题大面积采用了极具学术感的严肃衬线排版，在一个由黑白灰构成的空白虚空中，让用户自定义的强调色得以有跃然而出的清透感。

🚀 核心特性与功能
内容至上原则：摒弃复杂的动效装饰和臃肿小组件，将绝对的焦点还给文字。

参考刊物的排版优化：正文部分采用精心选择的衬线字体栈，提供沉浸式的阅读体验。

极简分类学：仅保留扁平化的网状“标签 (Tags)”系统，去除“分类”功能。

原生与高性能：纯原生 JavaScript，无 jQuery 等臃肿依赖。按需智能加载代码高亮与数学公式 (KaTeX) 引擎。

极速搜索：基于原生 JS 实现的轻量化客户端本地搜索功能。

沉浸式阅图：内置原生简易图片灯箱效果。

评论支持：支持 Giscus 评论系统整合。

⚙️ 安装与使用
在你的 Hugo 站点根目录下执行：
```Bash
git submodule add [https://github.com/yourusername/hugo-theme-lucidvoid.git](https://github.com/yourusername/hugo-theme-lucidvoid.git) themes/lucidvoid
```

详细配置请参考本仓库 `exampleSite` 文件夹下的 `hugo.toml` 模板。
